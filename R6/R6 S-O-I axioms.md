# **S-O-I Framework: R6 Foundation Schema**

---

## **1. The Three Levels**

```
I = Individual level      (what people do)
O = Organizational level  (what the system can do)
S = Strategic level       (what the organization privileges)
```

with:

```
R6 = {S, O, I}
```

---

## **2. The Structure (Same at Each Level)**

### **Three Axes**

```
Axis 1: DIRECTION      (stability ↔ change)
Axis 2: COORDINATION   (autonomy ↔ interdependence)  
Axis 3: REALIZATION    (direct ↔ mediated)
```

### **Two Poles per Axis**

```
Each axis has two complementary poles:
  Pole a = Agentive (stabilizing/individualizing/direct)
  Pole b = Instrumental (transforming/collectivizing/mediated)
```

### **Result: 6 Dimensions per Level**

```
3 axes × 2 poles = 6 dimensions per level
6 dimensions × 3 levels = 18 total dimensions
```

### **Linguistic Differentiation (Halliday's Transitivity)**

```
Isomorphism is structural, not linguistic.
Each level has a distinct ontological nature that governs the
linguistic register used to name and describe its dimensions.

Level I  (Intentional agent):
  Participant: Actor / Behaver
  Process types: material, behavioural
  Validation: "The person [verb]..."

Level O  (Structured collective):
  Participant: Carrier
  Process types: relational-attributive, abstract material
  Validation: "The system [verb]..."

Level S  (Unitary entity):
  Participant: Token
  Process types: relational-identifying, mental
  Validation: "The organization [verb] itself on/toward..."
```

---

## **3. The Isomorphism (Same Pattern, Different Levels)**

|Axis|I (Individual)|O (Organizational)|S (Strategic)|
|---|---|---|---|
|**Direction**|Demonstrating responsibility / Demonstrating innovation|Sustaining coherence / Enabling transformation|Positioning on value / Positioning on adaptation|
|**Coordination**|Acting autonomously / Acting collaboratively|Coordinating autonomy / Structuring cooperation|Orienting toward delegated governance / Orienting toward integrative governance|
|**Realization**|Contributing through results / Contributing through process|Regulating production output / Integrating realization processes|Committing to direct realization / Committing to mediated realization|

**Key principle**: Same functional pattern at each level, different expression governed by distinct linguistic registers.

---

## **4. The Vertical Relations**

### **Emergence (bottom-up)**

```
I ->(emerges)-> O ->(emerges)-> S

Individual competencies -> enable -> Organizational capabilities
Organizational capabilities -> enable -> Strategic stances
```

**Non-reductive**: O ≠ sum of I, S ≠ sum of O

### **Constraint (bidirectional)**

```
I <=> O <=> S

Bottom-up: I limits what O can do, O limits what S can achieve
Top-down: S demands O capabilities, O demands I competencies
```

### **Coherence Requirement**

```
For the system to work:
  Strategic stance ≤ Organizational capability ≤ Individual competence
  
If mismatch -> predictable failure
```

---

## **5. The Horizontal Relations**

### **Balance Between Poles**

```
Healthy system: BOTH poles active on each axis

NOT: Either/Or (stability XOR change)
BUT: Both/And (stability AND change)

Pathology: One pole dominates, other atrophies
Pathology: Both poles atrophy
```

### **Independence of Dimensions**

```
All 6 dimensions at each level are independent:
- Knowing dimension 1 tells you nothing about dimension 2
- Each dimension must be evaluated separately
```

### Key Relations:

```
  <-> : Isomorphism (same function, different level)
  ↑ : Emergence (enables upward)
  ↓ : Constraint (demands downward)
  <=> = Both directions matter
```

Level 3 (S): These are **stances the organization takes strategically** (relational-identifying/mental processes, cf. Halliday's Transitivity).

## **7. Core Principles**

1. **Isomorphism**: Same structure at all three levels
2. **Emergence**: Higher levels emerge from lower (but aren't reducible)
3. **Constraint**: Each level both enables and limits adjacent levels
4. **Balance**: Both poles needed on every axis
5. **Independence**: All dimensions orthogonal (can be evaluated separately)
6. **Completeness**: The 18 dimensions cover all organizational functions
7. **Linguistic differentiation**: Each level uses process types and participant roles consistent with its ontological nature (Halliday's transitivity)

## **8. Simple Notation**

```
S, O, I           = Three levels
a₁, a₂, a₃        = Three axes  
pa, pb            = Two poles per axis
d₁...d₆           = Six dimensions per level

I -> O -> S         = Emergence direction
I <=> O <=> S         = Constraint is bidirectional
S ≅ O ≅ I         = Structural isomorphism
```

**That's it.** The foundation is:

- 3 levels (I, O, S)
- 3 axes each (direction, coordination, realization)
- 2 poles per axis (complementary tensions)
- Relations: emergence ↑, constraint ↕, isomorphism ≅

Everything else builds from this.

# Method

## **9. Diagnostic Order (Methodological, Not Causal)**

### **Fundamental Distinction**

```
Causal Order ≠ Diagnostic Order

Axes remain orthogonal regardless of examination sequence
```

---

### **Standard Diagnostic Sequence**

```
DIRECTION → REALIZATION → COORDINATION
```

**Rationale**:

- **DIRECTION**: Highest abstraction, orients investigation
- **REALIZATION**: Most observable, tests S-O-I coherence
- **COORDINATION**: Most complex, explains gaps

---

### **Alternative Sequence (Crisis Context)**

```
REALIZATION → COORDINATION → DIRECTION
```

**Rationale**: Address immediate failure before root causes.

---

## **10. Relation to Causal Axioms**

```
DIRECTION ⊥ COORDINATION ⊥ REALIZATION  (orthogonal axes)

DIRECTION → REALIZATION → COORDINATION  (diagnostic path)
```

The diagnostic sequence is an **investigation optimization**, not a causal chain.

---

## **11. Simple Notation**

```
→ (diagnostic): Sequential examination order
⊥ (causal):     Orthogonal independence

DIRECTION → REALIZATION  (examine in this order)
DIRECTION ⊥ REALIZATION  (causally independent)
```

---

**That's it.** The methodological foundation:

- Axes are examined sequentially for efficiency
- Axes remain causally independent
- Alternative sequences exist for specific contexts
